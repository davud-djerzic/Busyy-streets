# 🚗 Busy Streets

Busy Streets is a 2D educational and entertaining mobile game developed in Unity, where players solve traffic intersections by applying 7 different traffic rules. Players progress through levels, earn money, and purchase various cars that can be stored and managed in a personal garage.

---

## 🎮 Gameplay

- Solve intersections based on 7 unique traffic rules for safe driving  
- Progress through levels and earn money for successfully completed challenges  
- Purchase different cars using in-game currency  
- Manage vehicles in your personal garage and select your favorite car  
- Increasing difficulty as you advance through levels  

---

## 📱 Platform

- Android  
- Optimized for mobile devices

## 🛠️ Technologies

- C# (scripting)  
- 2D graphics and animations
